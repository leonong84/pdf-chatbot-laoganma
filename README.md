# pdf-chatbot-laoganma
Using LLM to understand your PDF and chat with it!
Uses pinecone and replicate, both require their own API.

Python libraries:
- streamlit
- replicate
- PyPDF2
- pinecone-client
- pypdf
- langchain
- pdf2image
- pytesseract
- Pillow 
- sentence-transformers
- pdfplumber

Required packages:
- poppler-utils
- tesseract-ocr
